# Настройка PHP + SQLite

## Установка и настройка

1. Установил чистый PHP на ПК.
2. В env винды прописать Path к php (в поиске можно найти)
3. В php.ini (корне, где находится php) добавить строки:
   ```ini
   extension=sqlite3
   extension=pdo_sqlite
   ```
4. Перезапустил сервер или консоль (если нужно).

## Проверка установки

```php
<?php
phpinfo(); // для проверки поддерживает ли php вообще sqlite
```

## Запуск встроенного сервера

1. Открыть консоль в корневой папке проекта (где лежит index.php)
2. Выполнить команду:
   ```bash
   php -S localhost:8000
   ```
3. Сервер запустится и будет доступен по адресу: http://localhost:8000

### Важные моменты:
- Встроенный сервер ищет `index.php` как точку входа
- Работает только для разработки, не для продакшена
- Поддерживает .htaccess (базовые правила)
- Автоматически перезагружает файлы при изменении

### Остановка сервера:
- Нажать Ctrl+C в консоли
- Или закрыть окно консоли
